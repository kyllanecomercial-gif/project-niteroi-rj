<h1 align="center"> 🏙️ Morar em Niterói </h1>

<p align="center">
Este projeto é um site desenvolvido com HTML e CSS, criado para apresentar os principais benefícios de morar em Niterói, Rio de Janeiro. A página reúne informações sobre qualidade de vida, praias, lazer, cultura, infraestrutura e mobilidade, destacando os diferenciais da cidade de forma organizada e intuitiva.
<br/>
<br/>
Durante o desenvolvimento, foram aplicados conceitos fundamentais de desenvolvimento Front-end, como estruturação semântica em HTML, estilização com CSS, utilização de pseudoclasses para interatividade, inserção de imagens, links, campos de entrada (input) e botões, proporcionando uma experiência de navegação mais agradável ao usuário.
</p>
<br/>
<hr/>
<h1 align="center">🚀 Tecnologias utilizadas </h1>
<p align="center">
HTML5 <br/>
CSS3 <br/>
Pseudoclasses ( :hover, :active, entre outras ) <br/>
Imagens <br/>
Links <br/>
Inputs <br/>
Buttons <br/>
</p>
<hr/>
<h1 align="center">🎯 Objetivo </h1>
<br/>
<p align="center">O objetivo deste projeto foi colocar em prática os conhecimentos adquiridos em HTML e CSS, desenvolvendo uma página visualmente agradável e funcional, enquanto apresenta informações relevantes sobre os benefícios de viver em Niterói.</p>
